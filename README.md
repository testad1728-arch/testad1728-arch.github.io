
# Auto Blog (AdSense-ready, fully automated)

> ملاحظة: لا تُشجّع على النقرات أبداً. كل شيء هنا للزيارات فقط مع التزام السياسات.

## خطوات سريعة (بدون تعقيد)
1) أنشئ مستودع GitHub جديد (عام) باسم مثلاً: `auto-blog`.
2) ارفع **كل** ملفات هذا المشروع كما هي.
3) فعّل GitHub Pages من إعدادات الريبو ليأخذ من فرع `gh-pages`.
4) افتح الملف `site.config.json` وعدّل:
   - `site_url` إلى رابط صفحاتك (مثل: `https://USER.github.io/auto-blog/`)
   - ضع `adsense_publisher_id` الخاص بك، ثم ارجع إلى `index.html` وفك تعليق سكربت AdSense.
5) (اختياري) عدّل `feeds.json` لإضافة/حذف مصادر RSS مسموحة. المحتوى يُلخّص آليًا مع رابط واضح للمصدر.
6) انتظر المهمة اليومية. كل يوم تتولد تدوينات جديدة تلقائياً وتُنشر.

## أين أماكن الإعلانات؟
- أعلى الصفحة (ad-top) ووسط الصفحة (ad-mid). كلها **معلّقة** حتى تضع معرف AdSense.
- التزم السياسات: بدون تشجيع على النقرات وبدون حيل.

## تخصيص الشكل
- عدّل الألوان والخطوط من `assets/style.css` و `site.config.json`.
- اللغات: زر تبديل AR/EN موجود تلقائيًا.

## تطوير لاحق
- يمكنك إضافة فئات/أقسام، أو تحسين الخوارزمية لتلخيص أذكى، أو استبدال RSS بمصادر أخرى لديك حقوقها.
- تقدر تغيّر الجدولة من ملف الـ Workflow.

## تنبيه مهم
- لا نضمن القبول في AdSense؛ التزم بالمحتوى القيّم والسياسات واحرص على تحسين الأداء والسرعة.
